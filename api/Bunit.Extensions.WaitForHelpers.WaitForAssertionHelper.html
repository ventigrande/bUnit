    WaitForAssertionHelper(IRenderedFragmentBase, Action, Nullable<TimeSpan>)

    Creates an instance of the WaitForAssertionHelper type, which will until the provided assertion passes (i.e. does not throw an exception), or the timeout is reached (default is one second).

    The assertion is attempted initially, and then each time the renderedFragment renders.
