    WaitForStateHelper(IRenderedFragmentBase, Func<Boolean>, Nullable<TimeSpan>)

    Creates an instance of the WaitForStateHelper type, which will wait until the provided statePredicate action returns true, or the timeout is reached (default is one second).

    The statePredicate is evaluated initially, and then each time the renderedFragment renders.
