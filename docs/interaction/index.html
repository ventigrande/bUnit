    Interacting with a Component Under Test

    This section covers the various ways to interact with a component under test, e.g. trigger event handlers. This - and more - is covered in the following sub sections:

    • Triggering Event Handlers in Components: This covers how to invoke event handlers bound to elements rendered by a component under test.
    • Triggering a Render Life Cycle on a Component: This covers how to manually trigger a render cycle for a component under test.
    • Awaiting an Asynchronous State Change in a Component Under Test: This covers how to await one or more asynchronous changes to a component under test's state before continuing the test.
